## 如何搭建一个hexo博客

搭建博客之前需要安装hexo，hexo依赖于node和git，因此要确保系统上有这两个工具。

一个小坑：node版本不能太高，否则执行 hexo s 时会报 circular dependency的Warning，这时博客是打不开的。推荐安装node的长期支持版

接下来就是敲几个指令

$ npm install -g hexo-cli
$ hexo init blog
$ cd blog
$ npm install

创建后这是blog文件夹下的目录
